# busca_cep
Extensão para o Google Chrome, para buscar Cep

Instruções para instalação:

* Faça o download de todos arquivos.
* Abra o Google Chrome e vá para chrome://extensions/.
* Ative o modo de desenvolvedor (geralmente um botão no canto superior direito).
* Clique em "Carregar sem compactação" e selecione a pasta extraída.
